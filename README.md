# Vida Lírica - Landing Page

Este projeto consiste no desenvolvimento da Landing Page para a marca **Vida Lírica**, focado em transmitir calma, acolhimento e conexão com as raízes. Este arquivo serve como o guia oficial de implementação técnica para desenvolvedores Android/Web.

---

## 🎨 1. Paleta de Cores

As cores foram selecionadas para evocar sentimentos naturais e terrosos.

| Cor | HEX | Uso Principal |
| :--- | :--- | :--- |
| **Verde Musgo** | `#3E6B48` | Títulos principais, botões primários, destaques |
| **Verde Claro** | `#A8D5BA` | Hover de botões, blocos secundários, ícones de fases |
| **Bege Claro** | `#F5F0E6` | Fundos de seções de texto e benefícios |
| **Marrom Terra** | `#7B5E57` | Subtítulos, textos secundários, linhas divisórias |
| **Lilás Suave** | `#C6B5D1` | Fundo de bloco sobre mentora ou inspiração |
| **Branco** | `#FFFFFF` | Texto sobre fundos escuros, CTA secundário |

> **Dica:** Cores naturais e terrosas transmitem calma, raízes e acolhimento.

## 📐 3. Estrutura de Layout e Blocos

### Espaçamentos Padrão
* **Seções:** 60px (Top/Bottom)
* **Colunas Internas:** 30px gap
* **Padding Interno:** 20px – 40px

### Mapa Visual da Página
1.  **Hero:** Promessa + CTA (Fundo com imagem de mulher contemplando a roça).
2.  **Dor / Identificação:** Lista de dores + Imagem reflexiva.
3.  **Problema / Transformação:** Mensagem curta + Ilustração de raízes/árvore.
4.  **Método / Diferencial:** 4 fases em colunas com ícones (Corpo, Emoção, Mente, Espírito).
5.  **O que muda:** Lista de benefícios + Fotos pequenas de mulheres em ação.
6.  **Como funciona:** Timeline detalhada do processo.
7.  **Oferta:** Bloco em destaque (Bege Claro com sombra), bônus e investimento.
8.  **Pra que é:** Público-alvo e restrições.
9.  **Mentora:** Foto natural com tom acolhedor + Bio.
10. **Fechamento:** Reforço final em fundo Verde Musgo.

## 🔳 4. Componentes e Interações

### Botões (CTA)
* **Primário:** Verde Musgo, grande e centralizado.
    * *Hover:* Clarear para Verde Claro.
* **Secundário:** Pequeno, repetido nos blocos intermediários.
    * *Link:* Âncora para `#aplicacao`.
* **Texto:** "Quero aplicar para a Imersão Vida Lírica".

### Ícones Sugeridos
* 🌿 **Corpo:** Folha, movimento ou respiração.
* 💙 **Emoção:** Coração, onda ou nuvem leve.
* 💡 **Mente:** Lâmpada, seta ou caminho.
* 🌳 **Espírito:** Árvore, raiz ou sol.

### Animações (Opcional)
* Fade-in ou slide-in ao rolar a página.
* Hover suave (transition 0.3s) em todos os botões.
* Scroll suave para ancoragem de links.
